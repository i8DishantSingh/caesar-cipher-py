The python file (main.py) contains the implementation of caesar cipher encryption and decryption.

To the use the caesar cipher encryption and decryption in your projects, follow steps below:
  1. Download the 'main.py' file
  2. move the file to your prject directory
  3. import the content of the file to your project
  4. use the 'caesar_encrypt' function for encryption
  5. use the 'caesar_decrypt' function for decryption
